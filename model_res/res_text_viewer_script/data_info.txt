字幕数据约定

[csv]
txt:{
	pos : (str) {txt,wait,clear}
	voice : (str) -> (voice_str)
	text_code : '' --> 使用[BBcode]
 
	time_sleep : (double) 展示前睡眠时间
	time_apply : (double) 无对应语音时存续时间

	effect_out : (int) -1 0 1
	effect : (int) 对应效果id
}

关于动态效果(e.g. 倒计时) 由外挂的effect节点完成

[csv]
effect:{
	type: str
	rule: (str)
}

[Resource]
txt_pack{
	txt: [csv]
	effect: [csv]
}
